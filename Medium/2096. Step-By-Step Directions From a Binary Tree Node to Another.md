`Binary Tree`

You are given the `root` of a **binary tree** with `n` nodes. Each node is uniquely assigned a value from `1` to `n`. You are also given an integer `startValue` representing the value of the start node `s`, and a different integer `destValue` representing the value of the destination node `t`.

Find the **shortest path** starting from node `s` and ending at node `t`. Generate step-by-step directions of such path as a string consisting of only the **uppercase** letters `'L'`, `'R'`, and `'U'`. Each letter indicates a specific direction:

- `'L'` means to go from a node to its **left child** node.
- `'R'` means to go from a node to its **right child** node.
- `'U'` means to go from a node to its **parent** node.

Return *the step-by-step directions of the **shortest path** from node* `s` *to node* `t`.

**Example 1:**

![](https://assets.leetcode.com/uploads/2021/11/15/eg1.png)

**Input:** root = [5,1,2,3,null,6,4], startValue = 3, destValue = 6
**Output:** "UURL"
**Explanation:** The shortest path is: 3 → 1 → 5 → 2 → 6.

**Example 2:**

![](https://assets.leetcode.com/uploads/2021/11/15/eg2.png)

**Input:** root = [2,1], startValue = 2, destValue = 1
**Output:** "L"
**Explanation:** The shortest path is: 2 → 1.

**Constraints:**

- The number of nodes in the tree is `n`.
- `2 <= n <= 105`
- `1 <= Node.val <= n`
- All the values in the tree are **unique**.
- `1 <= startValue, destValue <= n`
- `startValue != destValue`

###### Using BFS & DFS

```java
class Solution {
    HashMap<TreeNode, TreeNode> parentMap = new HashMap<>();
    TreeNode startNode = null;
    int startValue;
    String res = "";
    public String getDirections(TreeNode root, int startValue, int destValue) {
        this.startValue = startValue;

        inorder(root);
        bfs(startNode, destValue);

        return res;
    }

    void inorder(TreeNode root){
        if(root == null) return;
        if(root.val == startValue) startNode = root;

        if(root.left != null) parentMap.put(root.left, root);
        inorder(root.left);

        if(root.right != null) parentMap.put(root.right, root);
        inorder(root.right);
    }

    void bfs(TreeNode startNode, int target){
        Queue<Pair> queue = new LinkedList<>();
        HashSet<Integer> visited = new HashSet<>();

        if(startNode == null) return;

        queue.offer(new Pair(startNode, ""));
        visited.add(startNode.val);

        while(!queue.isEmpty()){
            int size = queue.size();
            for(int i=0; i<size; i++){
                Pair temp = queue.poll();

                if(temp.node.val == target){
                    res = temp.str;
                    break;
                }

                TreeNode tempNode = temp.node;

                // Left
                if(tempNode.left != null && !visited.contains(tempNode.left.val)){
                    queue.offer(new Pair(tempNode.left, temp.str+"L"));
                    visited.add(tempNode.left.val);
                }

                //Right
                if(tempNode.right != null && !visited.contains(tempNode.right.val)){
                    queue.offer(new Pair(tempNode.right, temp.str+"R"));
                    visited.add(tempNode.right.val);
                }

                //Up
                if(parentMap.containsKey(tempNode) && !visited.contains(parentMap.get(tempNode).val)){
                    queue.offer(new Pair(parentMap.get(tempNode), temp.str+"U"));
                    visited.add(parentMap.get(tempNode).val);
                }
            }
        }
    }
}

class Pair{
    TreeNode node;
    String str;

    public Pair(TreeNode node, String str){
        this.node = node;
        this.str = str;
    }
}
```

##### Using LCA (Lowest Common Ancestor)

```java
class Solution {
    public String getDirections(TreeNode root, int startValue, int destValue) {
        StringBuilder res = new StringBuilder();
        StringBuilder LCAToStart = new StringBuilder();
        StringBuilder LCAToDest = new StringBuilder();
        TreeNode lcaNode = LCA(root, startValue, destValue);
        findPath(lcaNode, startValue, LCAToStart);
        findPath(lcaNode, destValue, LCAToDest);

        for(int i = 0; i<LCAToStart.length(); i++){
            res.append('U');
        }
        res.append(LCAToDest);
        return res.toString();
    }

    boolean findPath(TreeNode root, int target, StringBuilder sb){
        if(root == null) return false;
        if(root.val == target) return true;

        sb.append("L");
        if(findPath(root.left, target, sb)) return true;
        sb.setLength(sb.length() - 1);

        sb.append("R");
        if(findPath(root.right, target, sb)) return true;
        sb.setLength(sb.length() - 1);

        return false;
    }

    TreeNode LCA(TreeNode root, int p, int q){
        if(root == null) return root;
        if(root.val == p || root.val == q) return root;

        TreeNode left = LCA(root.left, p, q);
        TreeNode right = LCA(root.right, p, q);

        if(left != null && right != null) return root;
        return left == null ? right : left;
    }

}
```
