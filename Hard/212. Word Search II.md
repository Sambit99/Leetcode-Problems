`Binary Tree`

Given an `m x n` `board` of characters and a list of strings `words`, return *all words on the board*.

Each word must be constructed from letters of sequentially adjacent cells, where **adjacent cells** are horizontally or vertically neighboring. The same letter cell may not be used more than once in a word.

**Example 1:**

![](https://assets.leetcode.com/uploads/2020/11/07/search1.jpg)

**Input:** board = [["o","a","a","n"],["e","t","a","e"],["i","h","k","r"],["i","f","l","v"]], words = ["oath","pea","eat","rain"]
**Output:** ["eat","oath"]

**Example 2:**

![](https://assets.leetcode.com/uploads/2020/11/07/search2.jpg)

**Input:** board = [["a","b"],["c","d"]], words = ["abcb"]
**Output:** []

**Constraints:**

- `m == board.length`
- `n == board[i].length`
- `1 <= m, n <= 12`
- `board[i][j]` is a lowercase English letter.
- `1 <= words.length <= 3 * 104`
- `1 <= words[i].length <= 10`
- `words[i]` consists of lowercase English letters.
- All the strings of `words` are unique.

#### Brute Force Code with DFS

---

```java
class Solution {
    List<String> wordList;
    List<String> res;
    public List<String> findWords(char[][] board, String[] words) {
        wordList = new ArrayList<>(Arrays.asList(words));
        res = new ArrayList<>();

        int row = board.length;
        int col = board[0].length;

        for(int i=0; i<row; i++){
            for(int j =0; j< col; j++){
                dfs(board, i, j, new String(""), new HashSet<>());
            }
        }
        return res;
    }

    public void dfs(char[][] board, int i, int j, String s, HashSet<String> set){
        int row = board.length;
        int col = board[0].length;

        if(i < 0 || i >= row || j < 0 || j >= col || set.contains(""+i+j)){
            return;
        }

        s += board[i][j];
        set.add(""+i+j);

        if(wordList.contains(s) && !res.contains(s)){
            res.add(s);
        }

        dfs(board, i+1, j, s, set);
        dfs(board, i-1, j, s, set);
        dfs(board, i, j+1, s, set);
        dfs(board, i, j-1, s, set);

        set.remove(""+i+j);

    }
}
```

#### Optimized Code with Trie Data Structure

---

```java
class Solution {
    Set<String> res;
    public List<String> findWords(char[][] board, String[] words) {
        TrieNode root = buildTrie(words);
        res = new HashSet<>();

        int row = board.length;
        int col = board[0].length;

        for(int i=0; i<row; i++){
            for(int j =0; j< col; j++){
                dfs(board, i, j, root);
            }
        }
        return res.stream().collect(Collectors.toList());
    }

    public void dfs(char[][] board, int i, int j, TrieNode p){
        int row = board.length;
        int col = board[0].length;

        if(i < 0 || i >= row || j < 0 || j >= col){
            return;
        }

        char ch = board[i][j];

        if (ch == '#' || p.next[ch - 'a'] == null) return;
        p = p.next[ch - 'a'];

        if (p.word != null) {
            res.add(p.word);
            p.word = null; // So duplicate words not to be put
        }


        board[i][j] = '#';

        dfs(board, i+1, j, p);
        dfs(board, i-1, j, p);
        dfs(board, i, j+1, p);
        dfs(board, i, j-1, p);

        board[i][j] = ch;
    }

    public TrieNode buildTrie(String[] words) {
        TrieNode root = new TrieNode();
        for (String w : words) {
            TrieNode p = root;
            for (char c : w.toCharArray()) {
                int i = c - 'a';
                if (p.next[i] == null) p.next[i] = new TrieNode();
                p = p.next[i];
            }
            p.word = w;
        }
        return root;
    }
}


class TrieNode {
    TrieNode[] next = new TrieNode[26];
    String word;
}
```
